🧠 TASK CMD
Claim a URL, pick a template, set a password. Your space is yours.

How to claim your space

1

Pick a number 1–50 and visit that URL — e.g. /4 or /23

2

Select one or more templates — tap to toggle. Pick 1, mix 5, or select all 14. Each becomes its own tab.

3

Set your owner password. Your space is now locked to you. Optionally set separate view and collaborator passwords via the 🔐 Sharing button.

e.g. clnuniversal.vercel.app/4  ·  /12  ·  /50
👑
3 Access Levels — 3 Separate Passwords
👑 Owner — your password · /8
Full access. See and edit all tabs. Manage sharing settings. No restrictions.
👁 View Only — view password · /8?view=1
Read-only access. Requires a separate view password you set — different from your owner password. Only shows tabs you mark as visible in View Mode settings.
👥 Collaborator — collab password · /8collaborator
Editable access. Different URL pattern (/8collaborator) + separate password. Only shows tabs marked visible for Collab. Per-tab: can be fully editable (✏️) or read-only (📖).
🔐
Sharing Setup (Owner Only)

From inside your space, tap 🔐 Sharing in the toolbar. You'll see a grid of your tabs with two columns:

View column — toggle each tab 👁 Show or 🔒 Hide for view mode visitors.
Collab column — toggle each tab Hidden / Visible / or Visible+Editable (✏️) for collaborators.

Set a View password and a Collaborator password — both separate from your owner password. Share the appropriate URL + password with whoever needs it.
